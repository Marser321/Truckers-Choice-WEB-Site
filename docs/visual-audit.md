# Auditoria visual final

## Sistema compartido

- Tipografia display: Manrope con jerarquia por peso, tamano y espaciado.
- Tipografia funcional: Inter para cuerpo, navegacion, formularios y acciones.
- Fondos: `SectionBackground` separa superficies cinematograficas, contextuales y funcionales, con densidad y transiciones configurables.
- Heroes interiores: imagen, posicion y fuerza de overlay se controlan desde `InteriorHero`.

## Tratamiento por familia

| Familia | Tratamiento |
| --- | --- |
| Home | Hero cinematografico; bloques narrativos con media; secciones secundarias con superficies route, local, document y surface. |
| Servicios | Hero cinematografico; recorrido y catalogo sobre red operativa; detalle con fondos route, document y functional/surface. |
| Paquetes | Hero y catalogo sobre rutas conectadas, con tarjetas elevadas y luces contextuales. |
| Ubicaciones | Fondo propio de red local en hero, indice y detalles; fotografia reservada para cada oficina. |
| About | Superficies local, route y surface alternadas para separar historia, valores y confianza. |
| Recursos | Fondo documental operativo en hero, catalogo y articulos; grilla tecnica de baja intensidad. |
| Contacto | Hero fotografico; formulario sobre superficie funcional limpia con grilla tenue. |

## Criterios aplicados

- Los titulos largos usan line-height abierto y tracking moderado para mantener separacion clara entre palabras.
- El ambar se reserva para palabras clave del hero principal, kickers, estados y acciones.
- Las superficies funcionales mantienen contraste alto y menos ruido visual.
- Las transiciones de fondo evitan cortes planos accidentales entre secciones.
